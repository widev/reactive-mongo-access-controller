accesscontroller.wrapper

AccessControlCollection

case class AccessControlCollection(collection: BSONCollection)(users: Users, userGroups: UserGroups, sessions: Sessions) extends Product with Serializable

The accesscontroller.wrapper.AccessControlCollection wraps the reactivemongo.api.collections.default.BSONCollection.

It permits to interact with MongoDb with permissions checking for each operations, so you don't have to handle it manually anymore.

The documentation below is more focused on how to handle access controls than how ReactiveMongo internally works. However this class API is very similar to the reactivemongo.api.collections.default.BSONCollection one. I invite you to checkout this link if you do not know well the ReactiveMongo API: http://reactivemongo.org/releases/nightly/api/index.html#reactivemongo.api.collections.default.BSONCollection
