AccessControllerIndexesManager

sealed case class AccessControllerIndexesManager(indexesManager: IndexesManager) extends Product with Serializable

The accesscontroller.wrapper.AccessControllerIndexesManager wraps the reactivemongo.api.indexes.IndexesManager.

The API is almost the same as the reactivemongo.api.indexes.IndexesManager API. I invite you to checkout this link if you do not know well the ReactiveMongo API: http://reactivemongo.org/releases/nightly/api/index.html#reactivemongo.api.indexes.IndexesManager
